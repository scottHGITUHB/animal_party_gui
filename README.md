# animal_party_gui

#### 介绍
{**猛兽派对钓鱼脚本**}
"猛兽派对钓鱼脚本" 是一个为游戏《猛兽派对》开发的自动化钓鱼辅助脚本。其核心目的是帮助玩家在钓鱼过程中减少重复手动操作，解放双手。

关键功能如下：

自动化钓鱼：通过代码逻辑判断是否启用自动化钓鱼。
图像识别：依赖图像处理在屏幕上识别钓鱼的关键图标或界面。
多分辨率支持：支持不同分辨率的屏幕适配。
配置管理：允许用户自定义脚本参数，并保存为配置文件。

快捷键交互：
按2键：启用自动钓鱼。
按3键：手动触发钓鱼。

#### 安装教程

1.  python3.8+
2.  下载主文件及识别图片
3.  或者下载封装好的exe文件

#### 使用说明

1.  适用不同分辨率的脚本，首次使用可以使用默认配置，可以用重新勾选工具，自定义钓鱼识别界面。
2.  按2键可以启用自动钓鱼，3键手动触发钓鱼
3.  可以根据自己的需要更改配置，点击保存配置将会保存配置，以便下次使用

#### 快速开始
1.下载main.py和pointer_template.png，放在同一文件夹内，即可使用
2.config是预设配置，你可以根据自己实际使用需求调整。
3.咬钩区域设置为鱼饵右上角数字位置，指针区域设置为中间钓鱼进度调位置
